Overview

Filter is used to write your own pandoc filter in Ruby. A Filter is almost always created and immediately executed via the run method as shown in the following examples:

Examples:

Identity filter

Paru::Filter.run do
    # nothing
end

Remove horizontal lines

Paru::Filter.run do
  with "HorizontalRule" do |rule|
      if rule.has_parent? then
          rule.parent.delete rule
      else
          rule.outer_markdown = ""
      end
  end
end

Class Method Details

.run(&block) ⇒ Object

Run the filter specified by block. In the block you specify selectors and actions to be performed on selected nodes. In the example below, the selector is “Image”, which selects all image nodes. The action is to prepend the contents of the image's caption by the string “Figure. ”.

Examples:

Add 'Figure' to each image's caption

Paru::Filter.run do
    with "Image" do |image|
        image.inner_markdown = "Figure. #{image.inner_markdown}"
    end
end

def filter(&block)
    @selectors = Hash.new
    @filtered_nodes = []
    @doc = document

    @doc.each_depth_first do |node|
        @filtered_nodes.push node
        instance_eval(&block)
    end

    puts @doc.to_JSON
end

def with(selector)
    @selectors[selector] = Selector.new selector unless @selectors.has_key? selector
    yield current_node if @selectors[selector].matches? current_node, @filtered_nodes
end
